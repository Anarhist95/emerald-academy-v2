```cadence
pub fun main(x: Int): Int {
   if (x < 10) {
      return 0;
   } else if (x < 20) {
      return 1;
   } else {
      return 2;
   }
}
```

# **If / Else**

O Cadence suporta instruções condicionais **`if`**, **`else if`** e **`else`**.
